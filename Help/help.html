                           ЦЕЛЬ ПРОГРАММЫ
================================================================================

Автоматизация процесса составления меню для столовой предприятия.
Обеспечение единого информационного пространства для технологов, бухгалтеров
и кладовщиков. Повышение эффективности управления процессами:
- Формирование технологических карт и рецептур блюд
- Расчёт себестоимости и калорийности блюд
- Составление ежедневного меню
- Учёт движения продуктов на складе

================================================================================
                     СИСТЕМНЫЕ ТРЕБОВАНИЯ
================================================================================

СЕРВЕР БАЗ ДАННЫХ:
- Операционная система: Windows Server 2016/2019/2022 или Windows 10/11 Pro
- СУБД: Microsoft SQL Server 2022 (Express или Standard)
- Оперативная память: от 4 ГБ
- Дисковое пространство: от 10 ГБ

КЛИЕНТСКИЕ РАБОЧИЕ МЕСТА:
- Операционная система: Windows 10/11
- Оперативная память: от 2 ГБ
- .NET Framework 4.8 или выше
- Разрешение экрана: 1280x720 и выше

================================================================================
                    УСТАНОВКА И НАСТРОЙКА СЕРВЕРА
================================================================================

1. Установка SQL Server 2022

   1.1. Скачайте дистрибутив SQL Server 2022 Express с сайта Microsoft
   1.2. Запустите установку, выберите "Базовая установка"
   1.3. В процессе установки запомните имя экземпляра (по умолчанию: SQLEXPRESS)
   1.4. Установите SQL Server Management Studio (SSMS) для управления БД

2. Создание базы данных

   2.1. Откройте SQL Server Management Studio (SSMS)
   2.2. Подключитесь к серверу (используйте Windows аутентификацию)
   2.3. Выполните скрипт создания базы данных (файл CreateDatabase.sql):

   -----------------------------------------------------------------------------
   CREATE DATABASE MenuStolovayaDB;
   GO

   USE MenuStolovayaDB;
   GO

   -- Далее выполните все скрипты создания таблиц из папки Database
   -----------------------------------------------------------------------------

3. Настройка сетевого доступа к БД

   3.1. Откройте "SQL Server Configuration Manager"
   3.2. В разделе "Сетевые подключения SQL Server" включите "TCP/IP"
   3.3. Перезапустите службу SQL Server
   3.4. Откройте порт 1433 в брандмауэре Windows

================================================================================
                    УСТАНОВКА КЛИЕНТСКОГО ПРИЛОЖЕНИЯ
================================================================================

1. Первая установка

   1.1. Запустите файл MenuStolovaya Setup.exe
   1.2. Следуйте инструкциям мастера установки
   1.3. После установки запустите программу

2. Настройка подключения к базе данных

   2.1. При первом запуске откроется окно "Настройка подключения"
   2.2. Заполните параметры подключения:

   -----------------------------------------------------------------------------
   Поле                | Описание                        | Пример
   --------------------|---------------------------------|----------------------
   Сервер SQL          | Имя сервера БД                  | .\SQLEXPRESS
   Имя базы данных     | Имя базы данных                 | MenuStolovayaDB
   Аутентификация      | Windows или SQL                 | Windows
   -----------------------------------------------------------------------------

   2.3. Нажмите "Проверить подключение" для проверки
   2.4. При успешной проверке нажмите "Сохранить"

3. Установка на дополнительные компьютеры

   3.1. Скопируйте папку с установленной программой
   3.2. При первом запуске укажите тот же сервер баз данных
   3.3. Либо используйте установщик на каждом ПК

================================================================================
                         ПРАВИЛА РАБОТЫ В СИСТЕМЕ
================================================================================

1. Все документы (технологические карты, калькуляции, меню) ведутся в хронологическом порядке
2. Запрещено изменять утверждённые документы без согласования
3. Утверждение цен на продукты производит только бухгалтер
4. Списание продуктов со склада производится только при проведении требования накладной
5. Копирование базы данных выполняется администратором ежедневно
6. При возникновении ошибок обратитесь к администратору системы

================================================================================
                    РОЛИ ПОЛЬЗОВАТЕЛЕЙ И ДОСТУП
================================================================================

1. АДМИНИСТРАТОР
   ---------------
   Назначение: Полное управление системой, пользователями и правами доступа.

   Доступные функции:
   - Управление пользователями (создание, редактирование, блокировка)
   - Назначение ролей пользователям
   - Просмотр всех данных системы
   - Резервное копирование базы данных

2. ТЕХНОЛОГ
   ----------
   Назначение: Разработка рецептур, технологических карт и меню.

   Доступные функции:
   - Управление продуктами и категориями
   - Управление блюдами и видами блюд
   - Создание и редактирование технологических карт
   - Составление рецептуры с расчётом ингредиентов
   - Формирование ежедневного меню
   - Автоматический расчёт калорийности блюд
   - Печать меню
   - Создание требования накладной для кладовщика

3. БУХГАЛТЕР
   -----------
   Назначение: Контроль себестоимости, утверждение цен и калькуляций.

   Доступные функции:
   - Утверждение технологических карт
   - Управление калькуляционными карточками
   - Утверждение и изменение цен продуктов
   - Контроль себестоимости блюд
   - Расчёт фудкоста и маржинальности

4. КЛАДОВЩИК
   -----------
   Назначение: Учёт движения продуктов на складе.

   Доступные функции:
   - Приход продуктов на склад (оприходование товара)
   - Расход продуктов со склада (списание)
   - Обработка требований накладных от технолога
   - Просмотр текущих остатков продуктов
   - Управление номенклатурой продуктов

================================================================================
                    ИНСТРУКЦИЯ ДЛЯ ТЕХНОЛОГА
================================================================================

1. РАБОТА С ПРОДУКТАМИ
   --------------------
   1.1. Перейдите на вкладку "Продукты"
   1.2. Для добавления продукта нажмите кнопку "Добавить"
   1.3. Заполните поля:
        - Артикул: уникальный код продукта (например, К001)
        - Наименование: полное название продукта
        - Категория: выберите из списка
        - Единица измерения: кг, л, мл, г, шт
        - Потери холодной обработки (%): при очистке/разморозке
        - Потери горячей обработки (%): при варке/жарке
        - Белки, Жиры, Углеводы (г/100г): для расчёта калорийности
        - Цена: закупочная цена за единицу
   1.4. Нажмите "Рассчитать" для автоматического расчёта калорийности
   1.5. Нажмите "Сохранить"

2. РАБОТА С БЛЮДАМИ
   -----------------
   2.1. Перейдите на вкладку "Блюда"
   2.2. Для добавления блюда нажмите кнопку "Добавить"
   2.3. Заполните поля:
        - Наименование блюда: краткое название
        - Полное наименование: подробное описание
        - Вид блюда: салат, суп, горячее, гарнир и т.д.
        - Выход стандартный (г): вес порции (например, 200)
        - Время приготовления (мин): ожидаемое время
   2.4. Нажмите "Сохранить"

3. РАБОТА С ТЕХНОЛОГИЧЕСКИМИ КАРТАМИ
   ----------------------------------
   3.1. Перейдите на вкладку "Тех. карты"
   3.2. Нажмите "Добавить" для создания новой карты
   3.3. Выберите блюдо из выпадающего списка
   3.4. Укажите выход блюда (можно нажать "Рассчитать")
   3.5. Заполните технологию приготовления
   3.6. В панели "Состав рецептуры" добавьте ингредиенты:
        - Выберите продукт из списка
        - Укажите количество в КИЛОГРАММАХ (например, 0.200 для 200 г)
        - Укажите порядок закладки
        - Нажмите "Добавить"
   3.7. При необходимости меняйте порядок ингредиентов кнопками "Вверх"/"Вниз"
   3.8. Нажмите "Сохранить"

   ПРИМЕЧАНИЕ: Калорийность и выход блюда рассчитываются АВТОМАТИЧЕСКИ!

4. РАБОТА С МЕНЮ
   --------------
   4.1. Перейдите на вкладку "Меню"
   4.2. Нажмите "Добавить" для создания нового меню
   4.3. Выберите дату меню
   4.4. Нажмите "Блюда" для добавления блюд в меню
   4.5. В открывшемся окне:
        - Выберите блюдо из списка
        - Укажите количество порций
        - Укажите выход на порцию (г)
        - Укажите время подачи (Завтрак, Обед, Ужин, Полдник)
        - Нажмите "Добавить"
   4.6. После добавления всех блюд нажмите "Сохранить и закрыть"
   4.7. Для печати меню нажмите "Печать меню"

5. СОЗДАНИЕ ТРЕБОВАНИЯ НАКЛАДНОЙ
   ------------------------------
   5.1. Выберите меню в списке
   5.2. Нажмите кнопку "Требование накладная"
   5.3. Система автоматически создаст требование на основе рецептур
   5.4. Требование отправляется кладовщику для выполнения

================================================================================
                    ИНСТРУКЦИЯ ДЛЯ БУХГАЛТЕРА
================================================================================

1. УПРАВЛЕНИЕ ЦЕНАМИ ПРОДУКТОВ
   ----------------------------
   1.1. Перейдите на вкладку "Управление ценами продуктов"
   1.2. В списке продуктов выберите нужный продукт
   1.3. Нажмите "Изменить цену" для обновления цены
   1.4. После изменения цены нажмите "Утвердить цену"
   1.5. Утверждённая цена будет использована в калькуляциях

2. УТВЕРЖДЕНИЕ ТЕХНОЛОГИЧЕСКИХ КАРТ
   ---------------------------------
   2.1. Перейдите на вкладку "Технологические карты"
   2.2. Выберите карту со статусом "Черновик"
   2.3. Нажмите "Просмотр деталей" для проверки
   2.4. Если всё верно, нажмите "Утвердить техкарту"
   2.5. После утверждения автоматически создаётся калькуляционная карточка

3. УПРАВЛЕНИЕ КАЛЬКУЛЯЦИОННЫМИ КАРТОЧКАМИ
   ---------------------------------------
   3.1. Перейдите на вкладку "Калькуляционные карточки"
   3.2. Выберите карточку для просмотра
   3.3. Нажмите "Просмотр калькуляции" для деталей
   3.4. Для утверждения карточки нажмите "Утвердить"
   3.5. При необходимости можно отправить карточку на пересмотр

================================================================================
                    ИНСТРУКЦИЯ ДЛЯ КЛАДОВЩИКА
================================================================================

1. ОБРАБОТКА ТРЕБОВАНИЙ НАКЛАДНЫХ
   -------------------------------
   1.1. Перейдите на вкладку "Требования накладные"
   1.2. В списке выберите требование со статусом "Ожидает"
   1.3. Нажмите "Просмотр" для проверки состава
   1.4. При наличии достаточного количества продуктов нажмите "Подтвердить"
   1.5. При недостатке продуктов нажмите "Отклонить" и укажите причину

2. ПРИХОД ПРОДУКТОВ НА СКЛАД
   --------------------------
   2.1. Перейдите на вкладку "Приход продуктов"
   2.2. Нажмите "Новый приход"
   2.3. Выберите склад получатель
   2.4. Добавьте продукты с указанием количества и цены
   2.5. Нажмите "Сохранить", затем "Провести"

3. РАСХОД ПРОДУКТОВ СО СКЛАДА
   ---------------------------
   3.1. Перейдите на вкладку "Расход продуктов"
   3.2. Нажмите "Новый расход"
   3.3. Выберите склад отправитель
   3.4. Добавьте продукты с указанием количества
   3.5. Нажмите "Сохранить", затем "Провести"

4. ПРОСМОТР ТЕКУЩИХ ОСТАТКОВ
   --------------------------
   4.1. Перейдите на вкладку "Текущие остатки"
   4.2. При необходимости выберите склад для фильтрации
   4.3. Для поиска продукта используйте строку поиска

================================================================================
                    ИНСТРУКЦИЯ ДЛЯ АДМИНИСТРАТОРА
================================================================================

1. УПРАВЛЕНИЕ ПОЛЬЗОВАТЕЛЯМИ
   --------------------------
   1.1. Войдите под учётной записью администратора
   1.2. Перейдите на вкладку "Управление пользователями"
   1.3. Нажмите "Добавить" для создания нового пользователя
   1.4. Заполните поля: логин, пароль, ФИО
   1.5. Выберите роль: Администратор, Технолог, Бухгалтер, Кладовщик
   1.6. При необходимости установите флаг "Блокировка"

2. РЕЗЕРВНОЕ КОПИРОВАНИЕ БАЗЫ ДАННЫХ
   ----------------------------------
   2.1. Откройте SQL Server Management Studio
   2.2. Подключитесь к серверу
   2.3. Выберите базу данных MenuStolovayaDB
   2.4. Правой кнопкой мыши → Tasks → Back Up...
   2.5. Укажите путь сохранения и нажмите OK

3. ВОССТАНОВЛЕНИЕ БАЗЫ ДАННЫХ ИЗ РЕЗЕРВНОЙ КОПИИ
   ---------------------------------------------
   3.1. Откройте SQL Server Management Studio
   3.2. Подключитесь к серверу
   3.3. Правой кнопкой мыши на папке Databases → Restore Database...
   3.4. Выберите "Device" и укажите путь к файлу резервной копии
   3.5. Нажмите OK

================================================================================
                    ОТВЕТЫ НА ЧАСТО ЗАДАВАЕМЫЕ ВОПРОСЫ
================================================================================

Вопрос: Что делать, если программа не подключается к базе данных?
Ответ: 1. Проверьте доступность сервера SQL (ping)
       2. Убедитесь, что SQL Server запущен
       3. Проверьте имя сервера и базы данных в настройках
       4. Откройте порт 1433 в брандмауэре

Вопрос: Как пересчитать калорийность всех блюд?
Ответ: В окне технолога на вкладке "Блюда" нажмите кнопку "Обновить калорийность"

Вопрос: Почему не отображается цена блюда?
Ответ: Для отображения цены необходимо:
       1. Утвердить цены на продукты (делает бухгалтер)
       2. Создать и утвердить калькуляционную карточку

Вопрос: Как добавить новый вид блюда или категорию?
Ответ: В окне технолога перейдите на соответствующие вкладки и нажмите "Добавить"

Вопрос: Можно ли редактировать утверждённые документы?
Ответ: Нет. Утверждённые документы нельзя редактировать.
       Для изменений необходимо отправить документ на пересмотр.
